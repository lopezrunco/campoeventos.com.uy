## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/campoeventos.com.uy.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Run app
    ```sh
    npm start
    ```
4. Build & deploy
    ```sh
    npm run deploy
    ```
